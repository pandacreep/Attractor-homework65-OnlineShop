1. Сущность категория. Каждый товар будет привязан к определенной категории. Категория в общем обозначает как товар будет использоваться. Примеры: Категория phones. Сюда будут относиться мобильные телефоны, смартфоны, стационарные телефоны. Категория Multimedia. Сюда будут относиться телевизоры, муз-центры, муз-инструменты. Поля: id, name - название категории

2. Сущность товар. То что будет продаваться в onlineshop. Будет иметь привязку к определенной категории. Поля: id, name - название товара, image - изображение товара, price - цена единицы товара, category_id - категория товара

3. Сущность клиент(customer). Сущность нужна для идентификации лица, покупающего товары. email будет использоваться для идентификации корзины клиента. Поля: id, email - почтовый адрес, password - пароль, fullname - имя клиента

4. Сущность корзина(basket). Сущность нужна для хранения списка покупаемых товаров до момента оформления заказа. Поля: products - список товаров, totalPrice - полная стоимость всех товаров в корзине

5. Сущность заказ(order). Сущность нужна для хранения информации о каждом заказе после того, как корзина была заполнена и заказ был оформлен. Поля: id, email - email клиента, оформившего заказ, order_date - дата оформления заказа, fullname - имя получателя заказа (может отличаться от имени клиента), phone - телефон получателя, address - адресс получателя, payment_type - тип платежа, amount - сумма заказа
